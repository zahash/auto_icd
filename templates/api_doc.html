Welcome to the documentation for the rest api (/icd)

For demonstration purposes, I will be using python and its requests package

import requests (pip install requests) and json packages. 


    import requests, json

Store the url in a variable 


    url = "https://icd-10-ai.herokuapp.com/icd"

There are four parameters. "age", "sex", "doctor", "input". Create a dictionary with these four parameters as keys and your desired values 


    d = {
            "age":"50-60",
            "sex":"MALE",
            "doctor":"UROLOGY",
            "input":"typhoid"
        }

post a request to the url and get the response 


    response = requests.post(url, data=d)

store the content of the response in a variable 


    content = rensponse.content

The content you got is in the form of a binary string. So, decode it into a dictionary to get the final data 


    data = json.loads(content.decode())
The whole code: 

    import requests, json

    url = "https://icd-10-ai.herokuapp.com/icd"
    d = {
            "age":"50-60",
            "sex":"MALE",
            "doctor":"UROLOGY",
            "input":"typhoid"
        }

    response = requests.post(url, data=d)
    content = rensponse.content
    data = json.loads(content.decode())
